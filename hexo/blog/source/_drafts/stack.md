# 개념
스택(stack)은 후입선출(後入先出), LIFO(Last In First Out) 구조로 먼저 들어간 자료는 밑으로 깔리고 나중에 들어간 자료는 위에 얹어지게 되며,
가장 마지막에 들어온(가장 위에 위치한) 자료가 가장 마지막에 먼저 나가는 구조이다.
자료를 넣는 것을 push, 자료를 빼내는 것을 pop이라 한다.
